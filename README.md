## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/genadi53/YelpCamp_v2
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app
   ```sh
    npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
